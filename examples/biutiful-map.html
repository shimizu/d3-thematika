Beautiful Thematic Map

🎨 全機能デモンストレーション

  • レイヤー: GeojsonLayer（地形・都市）、PointCircleLayer（都市人口）、GraticuleLayer（経緯線）、OutlineLayer（境界）
  • エフェクト: SVGフィルター（ドロップシャドウ、グロー）、グラデーション、パターン
  • データ: 世界地図、都市人口データ（100万人以上）
  • 機能: インタラクティブ操作、スタイル切り替え、投影法変更

Beautiful Thematic Mapの作成例

// Beautiful Thematic Mapの作成例

// データ読み込み
const worldData = await d3.json('./geojson/world.geojson');
const citiesData = await d3.json('./geojson/stats/cities_oio_over_1m.geojson');

// SVGエフェクトの定義
const defs = map.svg.append('defs');

// グローエフェクト
const glowFilter = defs.append('filter')
    .attr('id', 'glow')
    .attr('x', '-50%').attr('y', '-50%')
    .attr('width', '200%').attr('height', '200%');
    
// グラデーション定義
const oceanGradient = defs.append('radialGradient')
    .attr('id', 'oceanGradient')
    .attr('cx', '30%').attr('cy', '30%');

// レイヤー構成
const worldLayer = new Thematika.GeojsonLayer({
    data: worldData,
    attr: {
        "fill": 'url(#oceanGradient)',
        "filter": 'url(#glow)',
        "stroke": '#00ffff',
        "stroke-width": 0.5
    }
});

const citiesLayer = new Thematika.PointCircleLayer({
    data: citiesData,
    radius: d => Math.sqrt(d.properties.population) / 500,
    attr: {
        "fill": '#ff6b6b',
        "filter": 'url(#cityGlow)',
        "opacity": 0.8
    }
});

GeojsonLayer デモ

GeojsonLayerを使用してGeoJSONデータを地図上に表示します。 世界地図データを表示し、アウトライン、経緯線、スタイリング機能を確認できます。

使用データ:
  • データ: examples/geojson/world.geojson（世界地図データ）
  • レイヤー: GeojsonLayer（国境線）、OutlineLayer（投影法境界）、GraticuleLayer（経緯線）
  • 機能: 投影法切り替え
// GeojsonLayerの使用例

const geojson =  await d3.json("./geojson/world.geojson");

const width = 800;
const height = 600;

const projection = d3.geoEquirectangular()
    .fitExtent([[10, 0], [width-10, height]], geojson);

const map = new Thematika.Thematika({
    container: '#map',
    width: width,
    height: height,
    projection: projection
});

// GeojsonLayerインスタンスを作成
const worldLayer = new Thematika.GeojsonLayer({
    data: geojson,                
    attr: { 
        fill:(d,i)=> i%2 === 0 ? "#1a3d1f" : '#e0f7fa', 
        filter: shadowFilter.url(), 
        stroke: '#1a3d1f',
        strokeWidth: 0.8,
        opacity: (d,i) => 0.9,
    }
});


// OutlineLayerインスタンスを作成（クリップパス機能ON）
const outlineLayer = new Thematika.OutlineLayer({
    createClipPath: true,
    clipPathId: 'earth-outline-clip',
    attr: {
        fill: 'none',
        stroke: '#2c3e50',
        strokeWidth: 2,
        opacity: 0.8
    }
});

// GraticuleLayerインスタンスを作成
const graticuleLayer = new Thematika.GraticuleLayer({
    step: [10, 10], // 10度間隔
    attr: {
        fill: 'none',
        stroke: '#ddd',
        strokeWidth: 0.5,
        opacity: 0.6
    }
});

// レイヤーインスタンスを地図に追加
map.addLayer('graticule', graticuleLayer);
map.addLayer('outline', outlineLayer);
map.addLayer('world_new', worldLayer);

Line Text Layer デモ

📝 LineTextLayer機能

  • 背景地図: examples/geojson/world.geojson（世界地図）
  • パス沿いテキスト: SVG textPathを使用したラベル表示
  • 機能: ライン描画タイプ切り替え（直線/アーク/スムージング）、ラベル表示切り替え
  • データ例: 世界一周ルート、貿易ルートのラベル表示
  • ガイドライン: LineConnectionLayerとの併用でパス可視化

LineTextLayerの使用例

// LineTextLayerの使用例（GeoJSON形式）

// LineString形式のデータ（複数点を結ぶ1本のライン）
const lineStringData = {
    type: "Feature",
    geometry: {
        type: "LineString",
        coordinates: [
            [139.7, 35.6],  // 東京
            [116.4, 39.9],  // 北京
            [37.6, 55.7],   // モスクワ
            [2.3, 48.8],    // パリ
            [-0.1, 51.5],   // ロンドン
            [-74.0, 40.7]   // ニューヨーク
        ]
    },
    properties: { name: "世界一周ルート" }
};

// MultiLineString形式のデータ（複数のライン）
const multiLineStringData = {
    type: "FeatureCollection",
    features: [
        {
            type: "Feature",
            geometry: {
                type: "LineString",
                coordinates: [[139.7, 35.6], [151.2, -33.8], [72.8, 19.0]]
            },
            properties: { name: "アジア太平洋ルート" }
        },
        {
            type: "Feature",
            geometry: {
                type: "LineString",
                coordinates: [[-0.1, 51.5], [2.3, 48.8], [31.2, 30.0]]
            },
            properties: { name: "ヨーロッパ・アフリカルート" }
        }
    ]
};

// LineTextLayerインスタンスを作成
const textLayer = new Thematika.LineTextLayer({
    data: lineStringData,     // またはmultiLineStringData
    textProperty: 'name',     // テキスト取得元プロパティ
    lineType: 'arc',          // 'straight', 'arc', または 'smooth'
    arcHeight: 0.3,           // アークの高さ（arcタイプの場合）
    smoothType: 'curveBasis', // スムージング時のカーブタイプ
    startOffset: '50%',       // textPath上のテキスト位置
    textAnchor: 'middle',     // テキストアンカー
    fontSize: 16,             // フォントサイズ
    fontWeight: 'bold',       // フォントウェイト
    attr: {
        "fill": '#ff6b6b',
        "stroke": '#ffffff',
        "stroke-width": 3
    }
});

// LineConnectionLayerと併用でパス表示
const connectionLayer = new Thematika.LineConnectionLayer({
    data: lineStringData,
    lineType: 'arc',
    arcHeight: 0.3,
    endArrow: true,
    attr: {
        "stroke": '#ff6b6b',
        "stroke-width": 3,
        "opacity": 0.8
    }
});

// 地図に追加（ラインを先に、テキストを後に）
map.addLayer('connections', connectionLayer);
map.addLayer('line-texts', textLayer);

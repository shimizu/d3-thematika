Line Text Layer デモ

📝 LineTextLayer機能

  • データ: LineString形式のGeoJSONデータ（サンプル路線データ）
  • レイヤー: LineTextLayer（ライン上テキスト）、LineConnectionLayer（ライン）
  • 機能: テキスト位置、配置方向、フォント設定の変更
  • 特徴: ラインの方向に沿ったテキストの自動回転、位置指定、複数位置表示
16px

実装サンプル

LineTextLayerの使用例

// LineTextLayerの使用例

// LineString形式のサンプルデータを作成
const lineData = {
    type: 'FeatureCollection',
    features: [
        {
            type: 'Feature',
            properties: { text: '東京→大阪' },
            geometry: {
                type: 'LineString',
                coordinates: [[139.6917, 35.6895], [135.5023, 34.6937]]
            }
        }
    ]
};

const map = new Thematika.Map({
    container: '#map',
    width: 800,
    height: 600,
    projection: d3.geoMercator().fitExtent([[10, 10], [790, 590]], lineData)
});

// LineConnectionLayerでラインを描画
const lineLayer = new Thematika.LineConnectionLayer({
    data: lineData,
    attr: {
        stroke: '#333',
        'stroke-width': 2
    }
});

// LineTextLayerでライン上にテキストを配置
const textLayer = new Thematika.LineTextLayer({
    data: lineData,
    position: 'middle',          // 中点に配置
    placement: 'along',          // ライン沿いに配置
    attr: {
        fill: '#d32f2f',
        'font-size': '16px',
        'font-weight': 'bold'
    }
});

// 複数位置にテキスト表示する場合
const multiTextLayer = new Thematika.LineTextLayer({
    data: lineData,
    multiplePositions: [0.2, 0.5, 0.8],  // 20%, 50%, 80%位置
    placement: 'along',
    attr: {
        fill: '#d32f2f',
        'font-size': '14px',
        'font-weight': 'normal'
    }
});

// レイヤーを地図に追加
map.addLayer('line', lineLayer);
map.addLayer('text', textLayer);

高崎駅周辺建物データ（MCP取得）

MCPを使用してOpenStreetMapから高崎駅周辺の建物データを取得し、GeojsonLayerで表示します。 約290件の建物データを建物タイプ別に色分けして表示します。狭い範囲に絞り込みD3.js描画を最適化しました。

使用データ:
  • データ: examples/geojson/takasaki/mcp-buildings2.geojson（高崎駅周辺建物データ、289件）
  • レイヤー: GeojsonLayer（建物）、OutlineLayer（境界）、GraticuleLayer（経緯線）
  • 機能: 投影法切り替え、建物タイプ別色分け、透明度調整
  • データソース: OpenStreetMap via MCP

// MCPで取得した高崎駅周辺建物データの表示

const buildingsData = await d3.json("./geojson/takasaki/mcp-buildings.geojson");

const width = 800;
const height = 600;

const projection = d3.geoMercator()
    .fitExtent([[10, 10], [width-10, height-10]], buildingsData);

const map = new Thematika.Map({
    container: '#map',
    width: width,
    height: height,
    projection: projection
});

// 建物タイプ別の色分け関数
function getBuildingColor(d) {
    const building = d.properties.building;
    if (building === 'residential') return '#ff9800';
    if (building === 'commercial') return '#2196f3';
    if (building === 'industrial') return '#795548';
    if (building === 'house') return '#ffeb3b';
    if (building === 'hotel') return '#e91e63';
    if (building === 'school') return '#4caf50';
    return '#9e9e9e'; // デフォルト色
}

// GeojsonLayerインスタンスを作成
const buildingsLayer = new Thematika.GeojsonLayer({
    data: buildingsData,
    attr: {
        "fill": getBuildingColor,
        "stroke": '#333',
        "stroke-width": 0.2,
        "opacity": 0.8
    }
});

// レイヤーを地図に追加
map.addLayer('buildings', buildingsLayer);

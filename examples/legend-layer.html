LegendLayer デモ

LegendLayerを使用してD3スケールに基づく凡例を地図上に表示します。 序数、連続、閾値スケールに対応した自動凡例生成機能を確認できます。

機能:
  • データ: examples/geojson/world.geojson（世界地図データ）
  • レイヤー: GeojsonLayer（国境線）、LegendLayer（凡例）
  • 機能: スケール型切り替え、凡例位置切り替え
// LegendLayerの使用例

// 序数スケールの例
const colorScale = d3.scaleOrdinal()
    .domain(['都市', '農村', '工業地域'])
    .range(['#ff0000', '#00ff00', '#0000ff']);

const legendLayer = new Thematika.LegendLayer({
    scale: colorScale,
    position: { top: '10%', left: '80%' },
    title: '土地利用',
    orientation: 'vertical'
});

// 連続スケールの例
const populationScale = d3.scaleSequential(d3.interpolateYlOrRd)
    .domain([0, 1000000]);

const popLegend = new Thematika.LegendLayer({
    scale: populationScale,
    position: { top: '70%', left: '5%' },
    title: '人口密度',
    orientation: 'horizontal'
});

// 地図に追加
map.addLayer('legend', legendLayer);
map.addLayer('population-legend', popLegend);

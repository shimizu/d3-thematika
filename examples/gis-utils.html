🗺 GIS関連ユーティリティ機能

  • データ: examples/geojson/world.geojson（世界地図データ）
  • GIS関数: getBbox（境界ボックス）、getCentroid（中心点）、getBboxCenter（BBox中心）、expandBbox（BBox拡張）

GIS Utilities の使用例

// GIS Utilities の使用例
// 日本のデータを取得
const japan = worldData.features.find(f => f.properties.name === 'Japan');

// Bounding Box を取得
const bbox = Thematika.getBbox(japan);
// { minX: 122.93, minY: 24.04, maxX: 145.82, maxY: 45.52 }

// 中心点を取得（座標の平均）
const centroid = Thematika.getCentroid(japan);
// { x: 138.25, y: 36.20 }

// Bounding Box の中心を取得
const bboxCenter = Thematika.getBboxCenter(bbox);
// { x: 134.37, y: 34.78 }

// Bounding Box を拡張（20%のパディング）
const expandedBbox = Thematika.expandBbox(bbox, 0.2);

// その他の便利な関数
const dimensions = Thematika.getBboxDimensions(bbox);
// { width: 22.89, height: 21.48 }

const isValid = Thematika.isValidGeoJSON(japan);
// true

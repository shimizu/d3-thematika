Line Connection Layer デモ

🔗 LineConnectionLayer機能

  • 世界地図: examples/geojson/world.geojson（背景用）
  • 接続ライン: 動的に生成されるGeoJSONデータ（LineString/MultiLineString）
  • 機能: ライン描画タイプ切り替え（直線/アーク/スムージング）、矢印表示、投影法切り替え
  • データ例: 世界一周ルート（赤線）、貿易ルート（色分け破線）
0.3

実装サンプル

LineConnectionLayerの使用例

// LineConnectionLayerの使用例（GeoJSON形式）

// LineString形式のデータ（複数点を結ぶ1本のライン）
const lineStringData = {
    type: "Feature",
    geometry: {
        type: "LineString",
        coordinates: [
            [139.7, 35.6],  // 東京
            [2.3, 48.8],    // パリ
            [-74.0, 40.7],  // ニューヨーク
            [-122.4, 37.7]  // サンフランシスコ
        ]
    },
    properties: { name: "世界都市接続ライン" }
};

// MultiLineString形式のデータ（複数のライン）
const multiLineStringData = {
    type: "FeatureCollection",
    features: [
        {
            type: "Feature",
            geometry: {
                type: "LineString",
                coordinates: [[139.7, 35.6], [2.3, 48.8], [-74.0, 40.7]]
            },
            properties: { route: "東京-パリ-NY" }
        },
        {
            type: "Feature",
            geometry: {
                type: "LineString",
                coordinates: [[151.2, -33.8], [103.8, 1.3], [77.2, 28.6]]
            },
            properties: { route: "シドニー-シンガポール-デリー" }
        }
    ]
};

// LineConnectionLayerインスタンスを作成
const connectionLayer = new Thematika.LineConnectionLayer({
    data: lineStringData,  // またはmultiLineStringData
    lineType: 'arc',       // 'straight', 'arc', または 'smooth'
    smoothType: 'curveBasis', // スムージング時のカーブタイプ
    arcHeight: 0.3,        // アークの高さ（arcタイプの場合）
    startArrow: false,     // 開始点の矢印
    endArrow: true,        // 終了点の矢印
    arrowSize: 12,         // 矢印のサイズ
    attr: {
        "stroke": '#ff6b6b',
        "stroke-width": 2,
        "opacity": 0.8
    }
});

// 複数色のライン（関数による動的色指定）
const tradeRoutes = new Thematika.LineConnectionLayer({
    data: multiLineStringData,
    lineType: 'smooth',
    smoothType: 'curveBasis',
    endArrow: true,
    arrowSize: 8,
    attr: {
        "stroke": (d, i) => ['#4ecdc4', '#45b7d1', '#96ceb4'][i % 3],
        "stroke-width": 2,
        "opacity": 0.7,
        "stroke-dasharray": '5,3'
    }
});

map.addLayer('connections', connectionLayer);
map.addLayer('trade-routes', tradeRoutes);

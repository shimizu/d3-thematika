RasterLayer デモ

RasterLayerを使用して地図上にラスター画像を表示します。 Equirectangular投影法の場合は高速に描画し、その他の投影法では画像を変換して表示します。

テスト画像:
  • 画像: examples/img/africa_img.png
  • 範囲: 西経25.86°〜東経66.43°、南緯38.48°〜北緯41.48°（アフリカ地域）
  • Equirectangular以外の投影法では、画像サイズは1000×1000ピクセル以下に制限されます。
  • 投影法を変更して変換処理の動作を確認できます。
// RasterLayerの使用例（アフリカ画像）
const map = new Thematika.Map({
    container: '#map',
    width: 800,
    height: 400,
    projection: d3.geoEquirectangular()
});

// ラスター画像レイヤーの追加
const rasterLayer = new Thematika.RasterLayer('raster', {
    src: './img/africa_img.png',
    bounds: [-25.855061, -38.477223, 66.427949, 41.479176], // [west, south, east, north]
    style: { opacity: 0.7 }
});

// 投影法を設定
rasterLayer.setProjection(map.getProjection());
map.addLayer('raster', rasterLayer);
